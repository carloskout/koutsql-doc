Primeiros passos com KoutDB

Antes de comerçarmos a criar e executar instruções SQL, devemos primeiro informar para KoutDB uma conexão com Banco de Dados existente. Para isso, basta passarmos por parâmetro um objeto PDO para o método KoutDB::setPDO: 

    
        $pdo = new \PDO(...);
        KoutDB::setPDO($pdo);

Prontinho! Agora nós podemos começar a criar e executar instruções SQL. Veja como é fácil:

    
        $result = KoutDB::select('descricao', 'preco_compra')
        ->table('produto')
        ->exec()
        ->getAll();

O trecho de código acima demonstra uma consulta de todos os registros da tabela produto. No método select('descricao', 'preco_compra') informamos a lista de campos que queremos recuperar separados por vírgula. Em seguida, chamamos o método table('produto') para informarmos o nome da tabela que será feita a consulta SQL. Para executarmos a instrução, chamamos o método exec() e para recuperarmos os registros retornados pela consulta usamos o método getAll(). Existem outros métodos além de getAll() para recuperar o resultado que serão demonstrados no decorrer da documentação.

Para vermos a instrução SQL gerada pela API, basta executarmos o método sql() em vez do método exec(): 

    
        $sql = KoutDB::select('*')
        ->table('produto')
        ->sql();

        var_dump($sql);

        SQL Gerada:
            SELECT * FROM produto;
