Primeiros passos com DataB

Antes de comerçar a criar e executar instruções SQL, primeiro você deve informar para DataB uma conexão com Banco de Dados existente. Para isso, basta passar por parâmetro um objeto PDO para o método DataB::setPDO: 

    
        $pdo = new \PDO(...);
        DataB::setPDO($pdo);

Prontinho! Agora você pode começar a criar e executar instruções SQL. Veja como é fácil:

    
        $result = DataB::select('descricao', 'preco_compra')
        ->table('produto')
        ->exec()
        ->getAll();

O trecho de código acima demonstra uma consulta de todos os registros da tabela produto. No método select('descricao', 'preco_compra') informamos a lista de campos que queremos recuperar separados por vírgula. Em seguida, chamamos o método table('produto') para informarmos o nome da tabela que será feita a consulta SQL. Para executarmos a instrução, chamamos o método exec() e para recuperarmos os registros retornados pela consulta usamos o método getAll(). Existem outros métodos além de getAll() para recuperar o resultado que serão demonstrados no decorrer da documentação.

Caso queira ver a instrução SQL gerada pela lib DataB, basta executar o método sql() em vez do método exec(): 

    
        $sql = DataB::select('*')
        ->table('produto')
        ->sql();

        var_dump($sql);

        SQL Gerada:
            SELECT * FROM produto;

Ok. Então você viu uma simples demonstração de uso da lib DataB, para prosseguir aprendendo mais recursos avançados navegue pela documentação e divirta-se \O/.
