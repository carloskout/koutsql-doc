Primeiros passos com KoutDB

Antes de comerçarmos a criar e executar instruções SQL, devemos primeiro informar para KoutDB uma conexão com Banco de Dados existente. Para isso, basta passarmos por parâmetro um objeto PDO para o método KoutDB::setPDO. 

    
        $pdo = new \PDO(...);
        KoutDB::setPDO($pdo);

Prontinho! Agora nós podemos começar a criar e executar instruções SQL. Veja como é fácil:

    
        $result = KoutDB::select('*')
        ->table('produto')
        ->exec()
        ->getAll();

Para vermos a instrução SQL gerada, basta executarmos o método sql() em vez do método exec(): 

    
        $result = KoutDB::select('*')
        ->table('produto')
        ->sql();

        var_dump($result);

        SQL Gerada:
            SELECT * FROM produto;
