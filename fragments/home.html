KoutDB SQL Builder

KoutDB consiste em uma API que disponibiliza serviços para criação e execução de instruções SQL de forma fácil e prática. A criação dessa API surgiu da necessidade de aplicar meus conhecimentos na linguagem PHP. Claro que existem diversos frameworks no mercado que são muitos bons, no entanto KoutDB não foi criada para competir e sim para demonstrar meus conhecimentos em programação PHP e SQL.

Como instalar

API KoutDB pode ser instalada através do Composer, que é gerenciador de pacotes para a linguagem PHP.
Basta executar o comando: composer require carloskout/koutdb 1.*

Instalado? Ótimo! Agora você pode começar a aprender a usar a API através de Exemplos práticos.
