Documentação DataB

DataB é uma library que facilita a criação e execução de instruções SQL usando PDO (PHP Data Object).

Detalhes

Versão atual: 1.0.0
Licença: MIT

Como instalar

DataB pode ser instalada através do Composer, que é gerenciador de pacotes para a linguagem PHP.
Basta executar o comando: composer require carloskout/datab 1.*

Instalado? Ótimo! Agora você está pronto para dá os primeiros passos com lib DataB através de exemplos práticos.
